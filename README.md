# inStruct
React-Native Application that connects users to skilled instructors

## Getting Started
``` 
cd inStruct && npm install
```

## Running the app
Through the expo app (downloaded from the app store):
```
expo start
```

For Android:
```
expo start --android
```

For ios:
```
expo start --ios
```
